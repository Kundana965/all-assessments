problem statement:- There is a group of N friends who wish to enroll in a course together. The course has a maximum capacity of M students that can register for it. If there are K
other students who have already enrolled in the course, determine if it will still be possible for all the N friends to do so or not.

this problem is from VJudge- Loops- BD

import java.util.*;
public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int T=sc.nextInt();
        for(int i=0;i<T;i++)
        {
            int N=sc.nextInt();
            int M=sc.nextInt();
            int K=sc.nextInt();
            System.out.println((N<=(M-K))?"YES":"NO");
        }
        
    }
}
