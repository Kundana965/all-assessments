Problem Statement:- Ezio can manipulate at most X number of guards with the apple of eden.
Given that there are Y number of guards, predict if he can safely manipulate all of them.

this problem is from VJudge- Loops- BB

import java.util.*;
public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int T=sc.nextInt();
        for(int i=0;i<T;i++)
        {
            int X=sc.nextInt();
            int Y=sc.nextInt();
            System.out.println((X>=Y)?"YES":"NO");
        }
        
    }
}
